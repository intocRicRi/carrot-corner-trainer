CARROT CORNER Page layout — 4 no-scroll options (shown after answering)

Each box simulates one laptop screen. All are shown in the answered state — the goal is that the coach feedback appears without scrolling. Pick a direction.

A · Three columns — Table · Decision · Coach

Coach gets its own column (reserved from the start), so nothing shifts on answer. Trade-off: the table is smaller.

B · Coach under the table

Left column = table on top, coach below it (uses the empty space under the table). Right column = the decision. Table stays big.

C · Coach replaces the action panel

Two columns. Once answered, the hand-history shorthand is swapped for the coach tip in the same spot — no third column, table stays large.

D · Table on top, decision + coach below

Table as a centered banner up top, then options and coach side by side underneath. Good for wide, short screens.
